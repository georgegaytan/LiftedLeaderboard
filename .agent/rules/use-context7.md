# Context7 Usage

- **Library Documentation:** When you need to look up documentation or code examples for a library or framework, use the `context7` MCP server tools.
- **Tools:** Use `resolve-library-id` to find the library ID, and `query-docs` to get specific information.
- **Preference:** Prefer using Context7 over general web searches for library-specific information.
